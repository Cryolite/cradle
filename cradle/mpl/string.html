Struct template string

cradle::mpl::string — A wrapper of a compile-time string.

Synopsis

// In header: <cradle/mpl/string.hpp>

template<typename T, T... Chars> 
struct string {
};

Description

In this description, X denotes a type, ci denotes an rvalue of type X, and s denotes an rvalue of type string<X, c1, c2, ..., cN;>.

Requires

T should be either char, wchar_t, char16_t or char32_t.

sizeof...(Chars) + 1 should be less than the implementation quantity of template arguments in a template declaration[7].

Valid syntaxes

    Category
    Type.
    Returns
    std::array<X, N> 
  • string<X, c1, c2, ..., cN>::value
    Category

    Type

    Returns

[7] The quantity recommended in [implimits], ISO/IEC 14882:2011 is 1024.
