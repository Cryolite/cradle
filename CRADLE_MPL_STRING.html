Macro CRADLE_MPL_STRING

CRADLE_MPL_STRING — Creates cradle::mpl::string from a string-literal.

Synopsis

// In header: <cradle/mpl/string.hpp>

CRADLE_MPL_STRING

Description

Requires

Should be used only in the following form; 

CRADLE_MPL_STRING ( string-literal )

Any other forms of use are ill-formed; no diagnostic required. This form can be used only as simple-type-specifier.

Approximate expansion

CRADLE_MPL_STRING ( string-literal )

is equivalent to 

::cradle::mpl::string<
  ::std::remove_const<
    ::std::remove_reference<
      decltype(string-literal[0])>::type>::type,
  string-literal[0],
  string-literal[1],
  .....
  string-literal[N]>

where N is equal to sizeof(string-literal) / sizeof(string-literal).
